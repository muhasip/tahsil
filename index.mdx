## Ürününüz için tek müşteri API'si

Web ve mobil istemciler yalnız `https://api.tahsil.dev` adresini çağırır. Banka bağlantısı ayrıntıları ve hassas credential değerleri uygulama cevaplarına veya entegrasyon loglarına taşınmaz.

## Production API'yi güvenle deneyin

API referansındaki salt okunur operasyonlarda **Test Request** ile doğrudan `https://api.tahsil.dev` adresine istek gönderebilirsiniz. İstekler üçüncü taraf proxy üzerinden geçirilmez.

<Warning>
  Yalnız kısa ömürlü bir test access token'ı kullanın. Parola, refresh token veya banka credential'ı girmeyin. Veri değiştiren operasyonlar portalda canlı çalıştırmaya kapalıdır; bu sayfalarda yalnız kopyalanabilir request örnekleri gösterilir.
</Warning>
